Installation of ApPHP EasyInstaller


A new installation of ApPHP EasyInstaller is a very straight forward process:

Step 1. Uncompressing downloaded archive.

Uncompress the EasyInstaller script archive.
The archive will create a directory called "PHPEI_xxx".


Step 2. Uploading files.

Copy directory named install/ (including all sub-folders and files) into directory of your script. For example, if your script (program) is placed in directory named MyProgram - copy folder install/ into MyProgram/ directory.


Step 3. Linking files.

Then link your default file redirection to install/start.php to begin installation procedure.


Congratulations, you now have ApPHP EasyInstaller was installed and now ready to work!
Your next step is Getting Started!
